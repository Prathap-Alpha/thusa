    Setup

    0 · Install on your phone

    Add Thusa to your home screen so it opens full-screen like a real app. Pick your phone:

    🤖 Android — Samsung, Pixel, etc.
    1. Open this page in Chrome.
    2. Tap Install Thusa below — or Chrome ⋮ menu → Add to Home screen.

    No button? Use the Chrome ⋮ menu → Add to Home screen.

    🍎 iPhone / iPad
    1. Open this page in Safari — it must be Safari, not Chrome.
    2. Tap the Share button (the box with an ↑ arrow) at the bottom of Safari.
    3. Scroll down, tap Add to Home Screen, then Add.

    Then open Thusa from the new icon and finish sections 1–2 below inside the app.

    1 · AI brain (Google Gemini)

    Each person uses their own free key (30 sec): Get a free Gemini key → tap Create API key, copy it (starts with AIza), paste below.

    2 · Microsoft 365 calendar

    From your Azure app registration (see the setup guide — one-time job).

    You sign in once with Microsoft's own page — Thusa never sees or stores your password. Tick “Stay signed in” when Microsoft asks.
